# Eye Guard

A lightweight Windows application that enforces periodic eye breaks based on the 20-20-20 rule.

## Quick Start (Read this first)

### Requirements
- Windows
- Python 3.11+
- Add Python to PATH during installation

### Installation

1. Run `install.bat`
2. Double-click `eye_guard_2020.pyw`
3. (Optional) Run `startup.bat` to start Eye Guard automatically when Windows starts.

You're done!
